Yml para template de presentación tutos

```{markdown}
---
title: Tu título 
subtitle: tu nombre # puede ser en comillas o sin. 
author:   # Aquí van los nombres de los tutores o sinodales
  - name: Dr. A
    affiliations:
      - id: uni
        name: institucion
  - name: Dra. B
    affiliations:
      - name: FC-UNAM
  - name: Dra.C
    affiliations:
      - name: FC-UNAM
date: today
date-format: long 
format:
  revealjs: 
    theme: [default, custom.css, styles.scss]
    slide-number: c/t
    show-slide-number: all
    transition: fade
    background-transition: fade
    highlight-style: pygments
    #header-logo: logo-ibt/logo_ibt_600dpi.png 
    #footer: "Tutoral Numero__"
    width: 1280
    height: 920
    max-scale: 1.0
filters:
  - reveal-header
bibliography: references.bib
biblio-style: apa
code-wrap: auto
execute: 
  freeze: auto
---
```
